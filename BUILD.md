# Инструкция по сборке приложения

## Требования

- Node.js (версия 14 или выше)
- npm
- macOS (для сборки macOS приложения)

## Установка зависимостей

```bash
npm install
```

## Сборка приложения

### Сборка для macOS

```bash
npm run build:mac
```

Это создаст:
- DMG файлы для x64 и arm64 архитектур
- ZIP архивы для x64 и arm64 архитектур

### Сборка для текущей платформы

```bash
npm run build
```

## Результат сборки

После успешной сборки файлы будут находиться в папке `dist/`:
- `Screen Streaming-1.0.0.dmg` - установочный файл для Intel Mac (x64)
- `Screen Streaming-1.0.0-arm64.dmg` - установочный файл для Apple Silicon (arm64)
- `Screen Streaming-1.0.0-mac.zip` - архив для Intel Mac
- `Screen Streaming-1.0.0-arm64-mac.zip` - архив для Apple Silicon
- `Screen Streaming.app` - приложение (внутри архива или DMG)

## Запуск приложения

После сборки вы можете:
1. Открыть соответствующий DMG файл (для вашей архитектуры) и перетащить приложение в Applications
2. Или запустить приложение напрямую из папки `dist/mac/` или `dist/mac-arm64/`

## Примечания

- При первом запуске macOS может потребовать разрешение на запуск приложения
- Перейдите в Системные настройки → Безопасность и конфиденциальность → Разрешить
- Приложение требует разрешения "Запись экрана" в Системных настройках
- Приложение не подписано кодом (для локального использования)
- Для распространения рекомендуется подписать приложение с помощью Apple Developer ID
